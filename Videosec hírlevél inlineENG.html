True-WDR

 

Traditional digital recordings either have details in the bright parts or set the exposition so they can have detailed darker spots. With True Wide Dynamic Range, the camera uses different exposure for each line of the image then combines them with an advanced algorithm, resulting in an image with a wide dynamic range.

With short exposure we get an image with details in the light spots

In the case of WDR the combined images have details in dark and light parts of the image

With long exposure we get an image with details in the darker parts of the image

Smart functions

 

Smart Intrusion detection

The camera can differentiate between vehicles pedestrians and motorcycles/bicycles. The user can customize a zone so it only alerts to a specified object (car/person/bike) and set the delay, so it does not do anything until the object is in the zone for a predetermined time.

 

 

Voice Deterrent

The Camera can playback prerecorded messages on its speakers. With these you can deliver warnings if someone is in a prohibited or hazardous area or for rules they need to follow (wear a mask/helmet). In the case of security a human voice is a better deterrent than a siren.
